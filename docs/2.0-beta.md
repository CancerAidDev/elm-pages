# elm-pages 2.0 beta

## Setup Instructions
```shell
npx dillonkearns/elm-pages#2bbfc3985f494ca728f56e2cf7725965bae9f054 init elm-pages-demo
cd elm-pages-demo
git init
git submodule add -b static-files https://github.com/dillonkearns/elm-pages.git
npm install
npm start
```
